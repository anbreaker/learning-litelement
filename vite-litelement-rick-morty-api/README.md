- ✅ Github Pages
- ✅ ES Modules
- ✅ Github Actions

[Online Demo](https://rodydavis.github.io/vite-lit-element-rick-api/)

## Getting Started

Start the dev server:

`npm run start`

Create a production build:

`npm run build`

Serve the build directory:

`npm run serve`
